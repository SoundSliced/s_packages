# s_bounceable Example

This example app demonstrates usage of the `s_bounceable` Flutter package.

How to run:

```bash
# From the package root
cd example
flutter pub get

# Run on a connected device or emulator (Android/iOS)
flutter run

# Or run on web (no native platforms required)
flutter run -d chrome
```

The main entry point is in `lib/main.dart`.
